Instructies
Je start met het onderdeel Foundational Learning. Hiermee leer de basisconcepten van Data Science en de belangrijkste features van de software. Je werkt vanuit realistische business cases en met bijbehorende datasets. Binnen een paar uur doorloop je een volledige Data Science pipeline en heb je een goed begrip van ML.

Omdat de software volledig via een grafische interface kan worden bediend, heb je geen programmeerkennis nodig om ermee te kunnen werken. Het programma biedt wel uitgebreide interfaces met de belangrijkste talen voor data analyse (Python en R) en is daarmee tevens geschikt voor geavanceerde ML vraagstukken.
